# Atom WebPPL Support

WebPPL support with a live preview tool for Atom Editor.

Install:
```bash
git clone https://github.com/smihael/atom-webppl-preview
cd atom-webppl-preview
apm link
```

Toggle Live Preview:

Press `ctrl-shift-h` in the editor to open the preview pane.
